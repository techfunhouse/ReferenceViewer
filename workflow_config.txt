# Workflow Configuration for Replit

## Primary Workflows

- Start application: npm run dev
  Starts the current Express server (will be replaced with client-server setup)

## Recommended Workflows (for client-server architecture)

- Start server: cd server && npm run dev
  Starts the Express server standalone

- Start client: cd client && npm run dev
  Starts the Vite client standalone

- Start full app: concurrently "cd server && npm run dev" "cd client && npm run dev"
  Starts both client and server concurrently

- Install all: npm install && cd client && npm install && cd ../server && npm install
  Installs dependencies for root, client, and server
